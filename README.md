# 公务员考试 · 政治理论刷题网站

## 使用方法

### 本地预览
```bash
cd docs
python3 -m http.server 8080
```
然后打开 http://localhost:8080

### 部署到 GitHub Pages
1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选择 `/docs`
4. 保存后即可通过 `https://<username>.github.io/<repo-name>/` 访问

## 功能
- ✅ 选择题 + 填空题混合练习
- ✅ 按专题分类练习
- ✅ 做题进度自动保存（localStorage）
- ✅ 错题回顾模式
- ✅ 详细解析 + 挖坑提醒
- ✅ 做题统计和正确率
